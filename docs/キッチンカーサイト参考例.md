# キッチンカーサイト参考例

友達に見せる参考サイトは、「飲食店一般」ではなく、キッチンカーやフードトラックとして実際に使われているサイトを中心に見る。

ただし、日本の個人キッチンカーは、SHOP STOPなどの同じ型の店舗ページが多く、店ごとの世界観まで作り込まれた単独サイトは少ない。そのため、出店予定の整理は日本のキッチンカー例、サイト全体の見せ方は海外のフードトラック公式サイトも参考にする。

## 参考にする見方

* 店の看板メニューが一瞬で分かるか
* 出店予定や場所が探しやすいか
* InstagramなどSNSへの導線があるか
* イベント出店やケータリング相談の入口があるか
* 店ごとの雰囲気やこだわりが写真と言葉で伝わるか

## 1. 出店予定とイベント導線を見せたい

### Kogi BBQ

URL: https://kogibbq.com/

参考になる点:

* フードトラックとして、メニュー、出店予定、ケータリング相談の導線がある
* 出店予定に日付、場所、時間がまとまっている
* 場所から地図へ移動できる
* SNSへの導線があり、サイトとSNSの役割が分かれている

友達への説明例:

「出店場所が変わるキッチンカーなら、こういうふうに日付・場所・時間を見せると分かりやすい。Instagramで告知して、詳しい予定はサイトで見る流れにできる。」

取り入れるなら:

* 今週の出店予定
* Googleマップへのリンク
* Instagramリンク
* イベント出店相談への入口

費用感:

| こちらが手動更新 | 0円〜更新代行費 | 出店予定が少ないうちは低コスト |
| Googleカレンダー連携 | 0円〜 | 友達がカレンダーを更新できる |

## 2. メニュー写真を強く見せたい

### Cousins Maine Lobster

URL: https://www.cousinsmainelobster.com/

参考になる点:

* 看板メニューの写真が強く、何を売っているかすぐ分かる
* メニュー、出店場所、イベント出店、オンライン注文が整理されている
* フードトラックでも「ブランド感」がある
* 写真の使い方が大きく、食べたい気持ちを作っている

友達への説明例:

「写真でおいしそうと思ってもらうなら、この見せ方が参考になる。キッチンカーでも、看板メニューの写真を大きく出すだけで印象がかなり変わる。」

取り入れるなら:

* 看板メニューを最初に大きく見せる
* メニューごとに写真、名前、価格、短い説明を載せる
* 人気メニューを先に見せる
* Instagram写真とサイト写真の雰囲気をそろえる

費用感:

| やり方 | 月額目安 | 説明 |
| --- | ---: | --- |
| 写真とメニューを固定掲載 | 0円〜数百円程度 | 変更が少ないなら低コスト |
| 季節メニューを更新 | 0円〜更新代行費 | 変更のたびに差し替える |
| 自分でメニュー更新 | 700円〜2,000円程度 | 更新画面つきにする場合 |

## 3. おしゃれで今どきのフードトラック感を出したい

### Wafels & Dinges

URL: https://wafels.com/

参考になる点:

* フードトラック発のブランドとして、楽しく親しみやすい雰囲気がある
* 商品写真、店舗情報、ケータリング、イベント相談がまとまっている
* ブランドカラーや言葉づかいに個性がある
* 「ただ売る」より「この店の世界観」を見せている

友達への説明例:

「おしゃれだけど堅すぎない、楽しいキッチンカーの雰囲気を出すならこういう方向が参考になる。色、写真、言葉づかいをそろえると“その店っぽさ”が出る。」

取り入れるなら:

* 店名に合う色を決める
* 見出しやボタンの言葉を店の雰囲気に合わせる
* 写真を大きく使う
* メニュー紹介を少し遊びのある文章にする

費用感:

| やり方 | 月額目安 | 説明 |
| --- | ---: | --- |
| 1ページサイト | 0円〜数百円程度 | 世界観を作りつつ低コストで始める |
| お知らせ追加 | 0円〜更新代行費 | イベント出店や新メニューを載せる |
| 更新画面つき | 700円〜2,000円程度 | 自分でお知らせを投稿する場合 |

## 4. イベント出店やケータリング相談を受けたい

### The Halal Guys

URL: https://thehalalguys.com/

参考になる点:

* フードカート発のブランドとして、メニュー、店舗、ケータリング、フランチャイズ導線が整理されている
* 一般のお客さん向けと、ビジネス向けの導線が分かれている
* メニュー写真とブランドカラーが強く、覚えやすい

友達への説明例:

「通常営業の注文受付はいらなくても、イベントに呼びたい人の入口はあると便利。一般のお客さんにはメニューと出店予定、主催者には出店相談の入口を見せると分かりやすい。」

取り入れるなら:

* イベント出店相談ボタン
* 対応エリア
* 必要なスペースや電源の有無
* 相談フォームまたはInstagram DMへの導線

費用感:

| やり方 | 月額目安 | 説明 |
| --- | ---: | --- |
| Instagram DMへ誘導 | 0円 | 最初は一番簡単 |
| Googleフォーム | 0円〜 | 希望日時や場所を聞きやすい |
| 専用フォーム | 0円〜数千円程度 | サイト内で自然に見せられる |

## 5. 日本のキッチンカーの出店情報を参考にしたい

### トトノエル

URL: https://schedule.mellow.jp/ss_web/shops/6eSD5D

参考になる点:

* 曜日、場所、住所、時間が整理されている
* SNSやHPへの入口がまとまっている
* 複数の出店場所を見せるときの情報整理として使える

注意点:

このページは、店ごとのブランドサイトというより出店情報をまとめるページ。おしゃれさや世界観の参考ではなく、「出店予定をどう整理するか」だけを見る。

友達への説明例:

「日本のキッチンカーだと、出店予定はこういう形で整理されていることが多い。サイト全体のデザインは別で作り込みつつ、予定表はこのくらい分かりやすくする。」

## 機能別のまとめ

### Instagramをサイトに載せたい

費用感:

| やり方 | 月額目安 | 説明 |
| --- | ---: | --- |
| Instagramリンクだけ置く | 0円 | 一番シンプル。サイトからInstagramへ飛ばす |
| Instagram投稿をサイト内に表示 | 0円〜1,500円程度 | 外部サービス利用で費用が出る場合がある |
| Instagram運用も相談する | 月額5,000円〜 | 投稿文作成や告知導線の整理まで見る場合 |

説明例:

「Instagramは今まで通り使って、サイトにはInstagramへの入口を置ける。最初はリンクだけで十分。投稿をサイト内に表示する形もできるけど、外部サービスの都合で表示が変わることがあるから、まずはリンクで始めるのが安全。」

### 自分で出店予定を更新したい

費用感:

| やり方 | 月額目安 | 説明 |
| --- | ---: | --- |
| こちらが手動で更新する | 0円〜更新代行費 | サイト自体は低コスト。更新のたびに依頼してもらう |
| Googleカレンダーを使う | 0円〜 | 友達がカレンダーを更新すると、サイト側にも反映しやすい |
| 更新画面つきにする | 700円〜2,000円程度 | 自分でお知らせや出店予定を編集できる。管理の手間は増える |

説明例:

「出店予定をよく変えるなら、自分で更新できる形にした方が楽。ただ、最初から更新画面を作ると月額や管理の手間が増える。まずはGoogleカレンダーや手動更新で始めて、困ってから強化してもいい。」

### ブログやお知らせを投稿したい

費用感:

| やり方 | 月額目安 | 説明 |
| --- | ---: | --- |
| noteやInstagramに投稿してサイトからリンク | 0円 | 一番始めやすい。サイト更新は少なくて済む |
| こちらがサイトに記事を追加 | 0円〜更新代行費 | 見た目は整えやすいが、投稿ごとに作業が必要 |
| 自分で投稿できるお知らせ機能 | 700円〜2,000円程度 | 自分で記事を増やせる。更新画面の使い方を覚える必要がある |

説明例:

「お知らせは最初から必須ではない。まずはInstagramで告知して、サイトにメニューと出店予定を置く。出店実績やイベントの様子を残したくなったら、お知らせ機能を足すといい。」
