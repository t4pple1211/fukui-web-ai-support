# キッチンカーサイト制作ヒアリング項目

## 最初に聞くこと

* 店名
* 活動エリア
* 主な出店場所
* 主な客層
* 一番売りたいメニュー
* サイトを作る目的
* InstagramやLINEなど、今使っているSNS

## 掲載に必要な素材

### 写真

* キッチンカー外観
* メニュー写真
* 調理中の写真
* 出店中の雰囲気
* 店主の写真
* ロゴや看板の写真

スマホ写真でもよいが、明るい場所で撮った写真があると印象が良くなる。

### メニュー情報

* メニュー名
* 価格
* 簡単な説明
* 人気メニュー
* 季節限定メニュー
* アレルギーや注意事項が必要か

### 出店情報

* 出店予定の見せ方
* 更新頻度
* 雨天時の対応
* Googleマップを載せたいか
* イベント出店の相談を受けるか

### 問い合わせ方法

* Instagram DM
* LINE
* メール
* 電話
* Googleフォーム
* 専用フォーム

どれをメインにするか決める。

## 公開後の費用確認

専門用語ではなく、「どう使いたいか」で確認する。

### お店専用のURL

* お店の名前が入ったURLがほしいか
* 無料URLでもよいか
* 年額1,500円〜5,000円程度なら使いたいか

説明するときは、「お店専用の住所を持つかどうか」と伝える。

### サイトの使い方

どこまで自分で使いたいかで、月額費用が変わる。

| やりたいこと | 月額目安 | 説明 |
| --- | ---: | --- |
| まず見せるだけでよい | 0円〜数百円程度 | メニュー、出店予定、問い合わせ先を見せる |
| 自分で更新したい | 700円〜2,000円程度 | 出店予定やお知らせを自分で変えられる |
| イベント出店の相談を整理したい | 1,500円〜数千円以上 | 出店依頼フォームや管理機能を入れる |

### 更新方法

* 出店予定は誰が更新するか
* 更新頻度は週1回か、月1回か、イベントごとか
* 自分で更新できる画面が必要か
* 更新は毎回こちらに依頼する形でもよいか
* 更新代行が必要な場合、月額費用をどうするか

### 問い合わせ方法

* Instagram DMへ誘導するだけでよいか
* 無料のGoogleフォームでよいか
* LINEで受けたいか
* メールフォームを使いたいか
* お客様の名前や連絡先をどう管理するか

## できることの説明

### すぐできること

* スマホで見やすい1ページサイト
* メニューと価格の掲載
* 出店予定の掲載
* Instagramへのリンク
* 問い合わせ導線の設置
* 写真と文章の差し替え

### 後からできること

* ブログやお知らせ
* 出店カレンダー
* イベント出店相談フォーム
* LINE連携
* お客様の声
* Googleマップ埋め込み
* 管理画面

## 打ち合わせ時の説明例

このプロトタイプは完成版ではなく、「こういう見せ方ができる」という確認用です。

実際に作るときは、店名、写真、メニュー、価格、出店予定を本物に差し替えます。まずは1ページで作って、Instagramのプロフィールから飛べるようにすると、初めて見る人にも情報が伝わりやすくなります。

公開後に反応を見ながら、お客様の声、出店カレンダー、イベント出店相談フォームなどを追加していく形が現実的です。
