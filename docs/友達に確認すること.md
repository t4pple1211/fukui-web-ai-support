# 友達に確認すること

food truck Michelle のサイトを作る前に、友達からもらうもの、聞くこと、こちらから伝えることを整理する。

## まず伝えること

このサイトは、Instagramの代わりではなく、Instagramを見た人が迷わず情報を確認できる場所として作る。

最初から大きな機能を入れる必要はない。まずは以下を分かりやすく見せる。

* 何を売っているキッチンカーか
* 看板メニューは何か
* いつ、どこで出店しているか
* イベント出店を相談したい人はどう連絡すればよいか
* Instagramの雰囲気や投稿をサイト上でも見せる

## もらうもの

### 写真

できればスマホ写真でもよいので、以下がほしい。

* 看板バーガーの写真
* ガーリックシュリンプバーガーの写真
* Michelle Burgerの写真
* キッチンカー外観の写真
* 鉄板でパティを焼いている写真
* イベント出店中の写真
* ロゴ画像
* Instagram投稿で使っている画像

写真は、明るい場所で撮ったものがあるとサイトの印象がかなり良くなる。

### メニュー情報

* メニュー名
* 価格
* 簡単な説明
* 一番売りたいメニュー
* 季節限定メニューがあるか
* 売り切れやすいメニューがあるか
* アレルギーや注意書きが必要か

### 出店情報

* 今後の出店予定
* よく出店する場所
* 営業時間
* 雨天時の対応
* 最新情報はどこで告知しているか
* 出店予定をどのくらいの頻度で更新するか

### 連絡先・SNS

* Instagramアカウント
* 問い合わせを受けたい方法
* イベント出店依頼はInstagram DMでよいか
* メールアドレスを載せてもよいか
* 電話番号を載せるかどうか

## 聞くこと

### サイトの目的

1. このサイトを見た人に、まず何をしてほしい？
2. 一般のお客さん向けを強くしたい？イベント主催者向けも強くしたい？
3. Instagramから来た人に、何を確認してほしい？

### お店の見せ方

1. Michelleらしさはどんな雰囲気？
2. 赤・黄・緑の今の投稿イメージはそのまま使いたい？
3. アメリカン、ストリート、ポップ、イベント感のどれを強くしたい？
4. ロゴや投稿デザインで絶対に残したい要素はある？

### メニュー

1. 看板メニューはどれ？
2. 最初に見せたいバーガーはどれ？
3. メニュー写真は今ある？
4. 価格はサイトに載せてもよい？
5. メニューの説明文はこちらで整えてよい？

### 出店予定

1. 出店予定はサイトに載せたい？
2. 出店予定は誰が更新する？
3. 更新頻度は週1回、月1回、イベントごとのどれに近い？
4. 自分で更新できる画面が必要？
5. まずは手動更新やGoogleカレンダーでよい？

### イベント出店相談

1. 学校祭、会社行事、地域イベントの出店依頼を受けたい？
2. 出店可能エリアはどこまで？
3. 出店時に必要な条件はある？
4. 電源やスペースの条件はある？
5. 相談はInstagram DMで受けたい？Googleフォームで受けたい？

### 公開後の費用

1. お店専用のURLは必要？
2. 無料URLでもよい？
3. 月額費用はどのくらいまでなら大丈夫？
4. 最初は月額を抑えた1ページサイトで始めてよい？
5. 更新代行が必要なら、月額サポートにするか都度依頼にするか？

## こちらから説明すること

### サイトでできること

* Instagramのプロフィールから飛べるページを作れる
* メニュー、出店予定、イベント出店相談を1ページにまとめられる
* 実際の投稿や写真を使って、Michelleらしい見た目にできる
* 出店予定は、手動更新、Googleカレンダー、自分で更新できる画面のどれかを選べる
* イベント出店相談は、Instagram DM、Googleフォーム、専用フォームのどれかにできる

### 最初は入れなくてよいもの

* 通常注文の予約機能
* 決済機能
* 会員登録
* 大きな管理画面
* 複雑な予約管理

キッチンカーの場合、まず必要なのは「メニュー」「出店予定」「イベント出店相談」「Instagram導線」。

### 費用感

| やりたいこと | 月額目安 | 説明 |
| --- | ---: | --- |
| まず見せるだけでよい | 0円〜数百円程度 | メニュー、出店予定、問い合わせ先を見せる |
| 自分で更新したい | 700円〜2,000円程度 | 出店予定やお知らせを自分で変えられる |
| 出店相談を管理したい | 1,500円〜数千円以上 | 出店依頼フォームや管理機能を入れる |

お店専用URLを使う場合は、別で年額1,500円〜5,000円程度かかることが多い。

### 注意点

* 写真の雰囲気でサイトの印象が大きく変わる
* 価格や出店予定が変わるなら、更新方法を先に決めておく
* 個人の電話番号やメールを載せる場合は慎重に決める
* イベント出店相談で個人情報を受ける場合は、管理方法を決める
* まずは小さく作り、反応を見てから機能を足す方が無駄が少ない

## 友達に送る短い説明文

```text
サイトを作る前に、写真・メニュー・出店予定・連絡方法を確認したいです。

通常注文の予約機能や決済機能は、今はいらないと思っています。

まずは月額を抑えた形で作って、必要になったら自分で更新できる仕組みや出店相談フォームを追加できます。
```
