# SKILLS.md

## このプロジェクトで使う前提スキル

このサイトは、初心者でも保守しやすい静的サイトとして作成しています。

## HTML

- セクション単位でコンテンツを整理する
- 見出し階層を崩さない
- CTAボタンは目的地が分かるリンクにする
- フォーム項目には `name` を付け、将来の外部サービス接続に備える

## CSS

- 色や影などの共通値は `:root` のCSS変数で管理する
- スマホ表示を優先し、狭い画面では1列表示にする
- ボタンやフォームは指で押しやすいサイズにする
- 余白を詰めすぎず、読みやすさを優先する

## JavaScript

- モバイルメニューなど、必要最小限の動きだけを担当する
- フォーム送信処理は、接続先が決まってから実装する
- 複雑な処理を入れる場合は、関数名で役割が分かるようにする
