# SKILLS.md

## Required Skills for This Project

このプロジェクトでは、以下のスキルを活用して開発してください。

## 1. HTML

* セマンティックなHTMLを書く
* section、header、footer、nav、mainを適切に使う
* SEOを意識した見出し構造にする
* フォーム項目にはlabelを付ける
* アクセシビリティを意識する

## 2. CSS

* レスポンシブデザイン
* Flexbox / Grid
* CSS変数
* 余白設計
* ボタン・カード・料金表のデザイン
* スマホファースト設計

## 3. JavaScript

* ハンバーガーメニュー
* スムーススクロール
* FAQの開閉
* フォームの簡易バリデーション
* 将来的なAPI接続を想定した構成

## 4. SEO

* titleタグ
* meta description
* OGP設定
* h1〜h3の適切な構成
* 地域名「福井」を自然に含める
* サービス名を分かりやすく入れる

## 5. Copywriting

* お客様の悩みから文章を作る
* 技術ではなくメリットを伝える
* 「作れます」ではなく「こう役立ちます」と書く
* 問い合わせ前の不安を減らす
* 料金や流れを明確にする

## 6. Business Understanding

このサイトは単なるポートフォリオではなく、営業用サイトです。

重視することは以下です。

* 信頼感
* 問い合わせ導線
* 料金の分かりやすさ
* 地元感
* 誠実さ
* 継続サポートの提案

## 7. Maintenance

今後の修正をしやすくするために、以下を守ってください。

* クラス名は分かりやすくする
* CSSをセクションごとに整理する
* 不要なコードを残さない
* READMEに編集方法を書く
* 画像差し替え箇所を分かりやすくする

## 8. Deployment

最終的に以下で公開できるようにしてください。

* GitHub Pages
* Cloudflare Pages

README.mdには以下を書いてください。

* ローカルでの確認方法
* ファイル構成
* 公開手順
* 画像や文章の差し替え方法
* 問い合わせフォームの接続候補
