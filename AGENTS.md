# AGENTS.md

## Project Role

このプロジェクトは、福井県を中心にWeb制作・AI活用支援・業務効率化支援を行う個人事業向けの営業用ホームページです。

あなたはWeb制作アシスタントとして、以下の方針に従って開発してください。

## Main Goal

訪問者が以下の状態になることを目指します。

1. 何を提供している人かすぐ分かる
2. 信頼できそうだと感じる
3. 料金感が分かる
4. 問い合わせしやすい
5. 「この人に相談してみよう」と思う

## Target Users

* 福井県内の飲食店
* 中小企業
* 個人事業主
* 美容室・整体・サロン
* ITやAIに詳しくない事業者

## Tone and Writing Style

文章は以下を守ってください。

* 分かりやすい日本語
* 専門用語を使いすぎない
* 上から目線にしない
* 不安をあおりすぎない
* 「ホームページを作る」ではなく「集客・業務改善を支援する」という表現にする

## Design Principles

* 信頼感
* 清潔感
* シンプル
* スマホ優先
* 地方の中小企業にも伝わりやすい
* 若いエンジニアらしい誠実さ

## Development Rules

* HTML / CSS / JavaScript を基本とする
* 静的サイトとして動作すること
* Cloudflare Pages や GitHub Pages で公開しやすい構成にする
* 外部ライブラリは必要最小限
* CSSは読みやすく整理する
* JavaScriptは必要な機能だけにする
* フォーム送信処理は後から差し替えやすくする
* 画像は仮画像でよい
* 架空実績を使う場合は、実績ではなく「サンプル制作」と分かる表記にする

## Important Sections

必ず以下を含めてください。

* ファーストビュー
* サービス紹介
* 料金プラン
* 制作実績・サンプル制作
* 制作の流れ
* 自己紹介
* よくある質問
* お問い合わせフォーム
* フッター

## Avoid

以下は避けてください。

* 「必ず売上が上がる」などの断定表現
* 過度な煽り
* 実績がないのに実績があるように見せる表現
* 大企業向けの堅すぎる表現
* 派手すぎるアニメーション
* 読み込みが重い構成

## Future Expansion

将来的に以下を追加しやすい設計にしてください。

* ブログ
* 見積もりシミュレーター
* AIチャット
* 予約フォーム
* お客様の声
* 管理画面
* CMS連携
